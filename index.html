About This Track

Devacation Revamp is a creative design challenge where participants use AI tools and prompt engineering to redesign the Devacation Hackathon webpage, making it more visually appealing, engaging, and user-friendly while introducing an innovative feature.

What You'll Do:

  • Create a modern UI layout using AI-powered tools
  • Improve visual hierarchy and readability
  • Enhance user journey and navigation
  • Propose original features that improve the hackathon page

Evaluation Criteria:

  • Prompt Based (10 marks): Clarity and structure of AI prompts
  • Design Based (10 marks): Visual appeal and layout
  • UX Based (10 marks): User experience and navigation
  • Innovation (10 marks): Originality of new features
